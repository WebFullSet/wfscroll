# Usage

Implementing wfscroll.js into your project is easy.
<br>

Add script right before closing ``</body>`` tag:

```html
<script src="https://your_domain.com/js/wfscroll.js"></script>
<script>
    const WebAnim = new WFScroll({
        element: '.auto-animate'
    })
</script>
```
